# ソフトウェア工学 2026

### 本講義で学んだこと
Github Pagesによって、リモートレポジトリの内容をwebpageとして公開できる。また、HP更改はCI/CDによって自動化でき、Jekyllを使用することでmarkdownファイルを自動でhtml化することができる。

これによって開発者はmarkdownファイルを作成するだけでホームページが作れるようになる。

本講義では詳しい手順とファイルの設定を学んだ。
